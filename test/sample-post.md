# Welcome to the Sample Post

This is a sample blog post that demonstrates the frontmatter format supported by the Publish to Dev.to GitHub Action.

## Features Demonstrated

- **Title**: Extracted from frontmatter
- **Description**: Used as the article description
- **Draft Status**: This post is marked as draft
- **Tags**: Multiple tags are supported (up to 4)
- **Series**: This post belongs to a series
- **Cover Image**: A placeholder image is used

## Code Example

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('World');
```

## Markdown Support

The action preserves all your markdown formatting:

- Lists
- **Bold text**
- *Italic text*
- `Inline code`
- Links: [Dev.to](https://dev.to)

## Conclusion

This sample post helps you test the action before using it with your real content.
