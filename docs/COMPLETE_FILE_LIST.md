**action.yml**
- Defines action inputs, outputs, and runtime
- Specifies Node.js 20 as runtime
- Sets branding (icon and color)

**package.json**
- Lists dependencies (@actions/core)
- Defines build scripts
- Project metadata

**tsconfig.json**
- TypeScript compiler configuration
- Target: ES2022, Module: CommonJS
- Strict mode enabled

**src/index.ts**
- Main action logic
- Reads markdown files
- Parses frontmatter
- Publishes to Dev.to API

**New to this project?**
